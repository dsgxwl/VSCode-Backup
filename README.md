# VSCode-Backup
## 用户设置

```js
{
    // 主题
    "workbench.colorTheme": "One Dark Pro",
    // 编辑器字号
    "editor.fontSize": 18,
    // 编辑器行高
    "editor.lineHeight": 38,
    // 一个制表符等于的空格数
    "editor.tabSize": 2,
    // 编辑器字体
    "editor.fontFamily": "Source Code Pro, Source Han Sans, Consolas, 'Courier New', monospace",
    // 编辑器字体粗细
    "editor.fontWeight": "500",
    // MD预览字体
    "markdown.preview.fontFamily": " Source Code Pro, Source Han Sans, -apple-system, BlinkMacSystemFont, 'Segoe WPC', 'Segoe UI', 'HelveticaNeue-Light', 'Ubuntu', 'Droid Sans', sans-serif",
    // MD预览字号
    "markdown.preview.fontSize": 20,
    // 自动保存
    "files.autoSave": "afterDelay",
    // 文件图标主题
    "workbench.iconTheme": "Monokai Pro (Filter Octagon) Icons",
    // 开文件时尝试猜测字符集编码
    "files.autoGuessEncoding": true,
}
```

## 插件

颜色主题 : One Dark Pro

文件图标主题 : VSCode Great Icons
